# Core verb forms (all verbs)

# Polite paradigm

* [ ] ～ます／～ません（非過去）
* [ ] ～ました／～ませんでした（過去）
* [ ] ～ましょう（意向）
* [ ] ～まして（連用・接続）
* [ ] ～ますれば（条件）
* [ ] ～なさい（準命令）／～な（禁止・俗）

# Negative paradigm

* [ ] ～ない（否定）
* [ ] ～なかった（否定過去）
* [ ] ～なくて（否定テ形）
* [ ] ～ないで（「～せずに」＝without ～ing）
* [ ] ～なければ／～なきゃ（条件）
* [ ] ～なかろう（否定意向・文語）
* [ ] ～ず（文語否定）／～ずに（文語「～ないで」）

# Voice / ability / causation

* [ ] Potential（可能：一段＝～られる／口語短縮あり、五段＝～える）
* [ ] Passive（受け身・尊敬：五段＝～れる／一段＝～られる）
* [ ] Causative（使役：～せる／口語：～さす）
* [ ] Causative-passive（使役受け身：～せられる／～させられる）

# Te-form combos (aspect, result, sequence)

* [ ] ～ている（進行・習慣・結果状態）
* [ ] ～てある（結果の残存）
* [ ] ～ておく（準備）
* [ ] ～ていく／～てくる（状態変化の方向性）
* [ ] ～てしまう（完了・遺憾／口語：～ちゃう／～じゃう）
* [ ] ～てみる（試み）
* [ ] ～てほしい（依頼・希望）
* [ ] ～てください（依頼）
* [ ] ～ては いけない（禁止）
* [ ] ～ても いい（許可）
* [ ] ～た ほうが いい（助言）

# Desire / tendency / simultaneity / manner

* [ ] ～たい（話し手の願望）
* [ ] ～たがる（第三者の願望）
* [ ] ～がち（傾向）
* [ ] ～ながら（同時進行）
* [ ] ～方（～かた：方法）
* [ ] 複合動詞（～始める／～続ける など）

# Hearsay / appearance / supposition / intention

* [ ] ～そうだ（様態「見た目」）
* [ ] ～そうだ（伝聞）
* [ ] ～らしい（らしい推量）
* [ ] ～みたいだ（口語推量）
* [ ] ～はずだ（当然・予測）
* [ ] ～つもりだ（意図）
* [ ] ～べきだ（当然の義務）
* [ ] ～まい（否定意志・文語）
* [ ] ～だろう／～でしょう（推量）

# Nominalization & “こと／の” 用法

* [ ] ～の（口語的名詞化／強調）
* [ ] ～こと（抽象的名詞化：基本形）
* [ ] ことができる（能力表現）
* [ ] ことがある（経験／時々ある）
* [ ] ことにする／ことになる（決定／成り行き）

# Obligation / necessity idioms

* [ ] ～ないと（いけない／だめ）
* [ ] ～なくては（いけない）→ 口語：～なくちゃ
* [ ] ～なければ（いけない）→ 口語：～なきゃ
* [ ] ～しなくては／～しては いけない（禁止・義務の派生）

# Permission / giving & receiving with て-form

* [ ] （～て）あげる／くれる／もらう（授受表現）
* [ ] すみません＋～て（謝罪・依頼）

# 五段・一段の活用ポイント

* [ ] 五段：語尾別の活用列（～う／～つ／～る；～ぬ／～む／～ぶ；～く／～ぐ；～す）
* [ ] 一段：～いる／～える（※五段例外 あり）
* [ ] て／た への音便規則
  ‣ ～う・つ・る→ ～って／～った
  ‣ ～ぬ・む・ぶ→ ～んで／～んだ
  ‣ ～く→ ～いて／～いた（※行く→いって／いった）
  ‣ ～ぐ→ ～いで／～いだ
  ‣ ～す→ ～して／～した
* [ ] 特記事項：行く・問う・請う の不規則 て／た 形

# Irregular verbs

* [ ] する（全活用：して／した、しろ／せよ、すれば、しよう、できる など）
* [ ] 来る（くる）（全活用：きて／きた、こい、くれば、こよう、こられる など）
* [ ] 名詞＋する（サ変複合）

# 敬語（尊敬・謙譲・丁寧）

* [ ] 尊敬：お＋連用形＋になる／～れる（受け身形の尊敬用法）
* [ ] 謙譲：お／ご＋連用形＋する／致す
* [ ] 不規則敬語動詞（仰る・いらっしゃる・召し上がる・ご覧になる・伺う・参る・頂く など）

# 形容詞・形容動詞（参考）

* [ ] い形容詞：～い／～かった／～くない／～くなかった／～くて／～ければ／副詞形～く／（～かろう）
* [ ] な形容詞：～だ／～です／～だった／～ではない／連体～な／副詞～に／条件～であれば
* [ ] いい→よい の形の注意

# コピュラ（だ／です）

* [ ] だ／ではない／だった／だろう／なら（ば）／で（連用）
* [ ] です／ではありません／でした／でしょう／でしたら／でして
